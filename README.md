# Personal Dashboard

A lightweight browser-based dashboard built with pure HTML, CSS, and JavaScript for organizing daily essentials, spiritual study, and personal utilities in one place.

This project combines:

- Daily utility shortcuts
- Live weather
- News feeds
- Personal to-do management
- Spiritual resources and study pages
- A customized homepage experience

Designed as a personal start page for daily use.

---

## Features

### Dashboard Home
Main homepage includes:

- Live clock
- Current date
- Theme switch (light / dark)
- Searchable sidebar menu
- Daily inspirational quote rotation
- Personal shortcuts
- Live weather search
- News sections
- To-do list manager
